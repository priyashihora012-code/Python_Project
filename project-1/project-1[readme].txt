

## Description

This is a simple Python program that collects personal information from the user and displays it in a formatted way.

The program demonstrates the use of:

* Variables
* User Input
* Data Types
* Type Conversion
* `type()` function
* `id()` function
* Basic Calculations
* Output Formatting

---

## Features

* Takes user input for:

  * Name
  * Age
  * Height
  * Favourite Number
* Calculates approximate birth year based on age
* Displays:

  * Variable values
  * Data types
  * Memory addresses
* Prints a user-friendly summary
* Ends with a thank-you message

---

## Python Concepts Used

| Concept              | Description                     |
| -------------------- | ------------------------------- |
| `input()`            | Used to take user input         |
| `int()`              | Converts input into integer     |
| `float()`            | Converts input into float       |
| `type()`             | Displays data type              |
| `id()`               | Displays memory address         |
| Variables            | Stores user data                |
| Arithmetic Operators | Used for birth year calculation |

---

## Program Flow

1. Display welcome message
2. Collect user information
3. Process the data
4. Display details and summary
5. Show exit message
